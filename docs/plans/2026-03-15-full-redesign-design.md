# Phase 4: Full Visual Redesign

**Date:** 2026-03-15
**Status:** Approved
**Scope:** Landing page, logo, auth pages, dashboard, sidebar, all project pages, typography, animations
**Direction:** Warm, approachable, creative. No emoji. No em dashes.

---

## 1. Logo and Icon

New keystone shape (architectural wedge stone) with door/archway negative space. Earth on cream. Minimal, recognizable at 16px. Logo lockup: shape + "KEYSTONE" in Instrument Serif tracked out + tagline in DM Sans light.

## 2. Landing Page

Route: `/` for unauthenticated users. Six sections:
- Hero: full viewport, topographic pattern bg, large heading, two CTAs, architectural line illustration
- How It Works: three numbered cards (Define, Build, Move In)
- Features: 3x2 grid with Lucide icons
- Diaspora: dedicated section for overseas builders
- Pricing: four tiers (Foundation free, Builder $29, Developer $79, Enterprise contact)
- Footer: logo, links, copyright

## 3. Auth Pages

Split screen. Left: form on cream. Right: warm gradient with rotating quote. Larger inputs, full-width buttons, clay-colored links.

## 4. Dashboard

Greeting bar with name + date. Elevated project cards with market-color top border, Instrument Serif name, progress ring, mini sparkline. Designed empty state with illustration.

## 5. Sidebar

240px width, new logo, emerald active state, smooth hover transitions, collapsible to icon-only mode on desktop.

## 6. Page System

Consistent header strip (title + breadcrumb + action). Card accent dots. Pill badges. Rounded progress bars. Designed empty states with SVG illustrations.

## 7. Typography Scale

Hero 48px, Page 24px, Section 18px, Card 14px, Body 13px, Secondary 12px, Caption 11px, Data 13px mono, Tiny 10px uppercase.

## 8. Animations

Fade-in page transitions (200ms), card hover lift (150ms), button scale, sidebar slide, chart skeleton loading, form focus ring, toast slide-in. All subtle.
